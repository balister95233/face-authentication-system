# 🔐 Face Authentication System

A secure face authentication system using OpenCV with OTP verification for enhanced security.

---

## 🚀 Features
- Face Detection using OpenCV
- OTP Verification System
- Real-time Camera Access

---

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy

---

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the project:
python main.py
